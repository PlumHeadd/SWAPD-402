# SWAPD-402 - Tasky

A productivity Flutter application with task management and focus timer features.

## Features

- **Home Screen**: Dashboard with quick actions and stats
- **Tasks Screen**: Create, manage, and track tasks with priorities and due dates
- **Timer Screen**: Focus timer with customizable sessions

## Phase 1 Complete

All Phase 1 requirements implemented:
- Bottom navigation with 3 functional screens
- Responsive design across different screen sizes
- Clean UI with consistent color scheme and typography
- Pure Flutter/Dart implementation
